# My Personal Blog

This repository contains my personal blog, built using Jekyll with a custom theme gem.

## Table of Contents

1. [Rake Tasks](#rake-tasks)


## Rake Tasks

Rake is used for task automation. Install Rake with:

- **Installing Rake**
  ```sh 
  gem install rake
  ```

## Available Tasks

- **Create a New Post**
  ```sh
  rake post["Post Title"]
  ```
